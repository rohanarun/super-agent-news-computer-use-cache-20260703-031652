How Super’s computer-use cache works

1. Observe

Super agents watch real browser and desktop interactions: clicks, typing, navigation, and confirmations.

2. Verify

Successful runs are confirmed, audited, and scoped so unsafe actions are never blindly replayed.

3. Reuse

The validated action trace is saved into a computer-use cache for future runs.

4. Improve

Each repeat workflow gets faster and more predictable instead of starting from scratch.
